# Email Cleaner

A Thunderbird WebExtension for cleaning up your inbox by identifying and removing emails from repeat senders.

## Features

- **Sender summary view** — on load, scans the last 6 months of mail and displays a grouped list of senders with their message count and total size.
- **Message drilldown** — click any sender row to expand and see each individual message from that sender.
- **Open in Thunderbird** — click a message in the drilldown view to open it in the native Thunderbird client for review.

## Usage

1. Install the extension in Thunderbird.
2. Open the extension panel — it will automatically load a summary of senders from the past 6 months.
3. Click a sender row to see the individual messages.
4. Click a message to open it in Thunderbird.
5. Select messages to delete and clean up your inbox.
